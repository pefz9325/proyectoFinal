Sigue estos pasos para ejecutar el proyecto en tu computadora usando Visual Studio Code.

### 1. Requisitos ###
* Tener instalado **VS Code**.
* Tener instalado el **SDK de .NET** (puede descargarlo de la página oficial de Microsoft).
* Tener la extensión de C# Dev Kit instalada en VS Code.

### 2. Abrir el proyecto ###
1. Abre VS Code.
2. Ve a `Archivo` > `Abrir carpeta...` y selecciona la carpeta donde están todos los archivos del proyecto (`Inicio.cs`, `menu.cs`, etc.).
3. Tambien puede abrir directamente vs code y arrastrar la carpeta mi tiendita a vs code en el editor, le saldra un cuadro le damos 
a si y listo

### 3. Ejecutar el programa ###
Hay que hacerlo por la terminal:
1. Abre una terminal nueva en VS Code (`Terminal` > `Nueva terminal`).
2. Escribe el siguiente comando y presiona Enter:
 dotnet run
